Privacy Policy / プライバシーポリシー

App: JLPT Vocab Shorts ／ Last updated: 2026-06-28

English

This Privacy Policy explains how the App handles data. The App is a personal tool used solely by its owner to upload the owner's own educational videos to the owner's own TikTok account.

  1. Data we access. Through the TikTok Content Posting API, the App obtains an OAuth access token and refresh token for the owner's own TikTok account, with the video.upload scope only.
  2. How tokens are used. Tokens are used only to upload video files to the owner's TikTok inbox (drafts). They are not used for any other purpose.
  3. Storage. Tokens are stored locally on the owner's own computer and are never transmitted to any third party or external server operated by the App.
  4. No collection of other users' data. The App has no end users other than the owner and does not collect, store, sell, or share personal data of any other person.
  5. No third-party sharing. No data is shared with third parties, except the direct API communication with TikTok required to perform uploads.
  6. Retention & deletion. Tokens can be deleted at any time by removing the local token file. Revoking the App's access in TikTok settings invalidates the tokens.
  7. Contact. Inquiries can be directed to the owner via the GitHub account that hosts this document (github.com/myjapanesebitesbiz).

日本語

本プライバシーポリシーは、本アプリにおけるデータの取り扱いを説明します。本アプリは、運営者 本人が、運営者自身の学習用動画を運営者自身の TikTok アカウントへ投稿するためだけに使用する 個人用ツールです。

  1. アクセスするデータ. TikTok Content Posting API を通じて、運営者自身の TikTok アカウントの OAuth アクセストークンおよびリフレッシュトークンを、 video.upload スコープに限って取得します。
  2. トークンの用途. トークンは、運営者の TikTok の受信箱（下書き）へ動画 ファイルをアップロードする目的にのみ使用し、それ以外には使用しません。
  3. 保管. トークンは運営者自身のコンピュータ上にローカル保存され、本アプリが 運営する第三者・外部サーバーへ送信されることはありません。
  4. 他者のデータを収集しない. 本アプリには運営者以外の利用者は存在せず、他者の 個人データを収集・保存・販売・共有することはありません。
  5. 第三者提供なし. アップロード実行のために必要な TikTok との直接的な API 通信を除き、いかなるデータも第三者へ提供しません。
  6. 保持と削除. トークンはローカルのトークンファイルを削除することで、いつでも 削除できます。TikTok の設定で本アプリのアクセスを取り消すとトークンは無効になります。
  7. 連絡先. 本書を公開している GitHub アカウント （github.com/myjapanesebitesbiz）を通じてお問い合わせいただけます。
